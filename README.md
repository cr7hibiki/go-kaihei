# Go-Kaihei

> **警告：本项目仍处于开发状态！尚未发布 release 版本，目前无法使用**

Go-Kaihei 是一个使用 golang 封装的开黑啦机器人SDK. 该项目基于httpClient库[Resty](https://github.com/go-resty/resty) 开发
